## To do list App V : 0.1

> this is a simple to do app built with vanilla js

### App Capabilities

1. :large_orange_diamond: Add to do list.
2. :negative_squared_cross_mark: Delete from to do list.
3. :arrow_down_small:Search in to do list.

#### Tech Used

-  Html
-  Css
-  Vanilla JS
